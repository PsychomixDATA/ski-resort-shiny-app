# What Makes an Ideal Ski Resort?
This Shiny application helps users compare destination ski resorts around the world.

Features include:
• Explore resorts using filters
• Compare price, terrain, and vertical drop
• Personalized resort recommendations
• Detailed resort profiles

Dataset:
Original excel file of all destination ski resorts worldwide
Data preparation file in R
Cleaned data as a csv file
app.R shiny application file to run with resorts_cl

Created for DATA 824
